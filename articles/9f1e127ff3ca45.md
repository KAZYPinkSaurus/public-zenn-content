# 藤岡 様

このたびは、これまでOptFitのプロジェクトに深く関わっていただき、本当にありがとうございます。  
日々の業務の中で発揮してくださっている高い技術力と責任感に、心から感謝しています。

---

## これまでのご貢献について

藤岡さんには、今年の春から業務委託として参画いただき、

- コードベースの移行  
- Terraform apply の CD 化  
- ECS 化  
- マイグレーション自動化  
- FC ロール機能の実装  

など、**現在のOptFitにとって欠かすことのできない基盤システム**を次々と構築していただきました。  
いずれも難易度が高く、事業インパクトの大きい領域であり、藤岡さんの貢献なくして今の開発体制は成り立っていません。

現在取り組んでいただいている **「映像の不通自動通知機能」** も、  
技術的に非常にチャレンジングであると同時に、プロダクトの信頼性を大きく左右する重要な機能です。  
そのようなテーマを安心してお任せできること自体、私たちにとって大きな信頼の証だと感じています。

OptFitは今年、資金調達を行い、**シリーズBのフェーズ**に入りました。  
2025年は基盤を整える「守り」の一年でしたが、**2026年は明確に「攻め」の一年**になります。

事業を次のステージへ進めるためには、

- 高い機動力を持ち  
- 技術面で意思決定を担い  
- チームを率いていける  

**キーパーソンの存在が不可欠**です。
